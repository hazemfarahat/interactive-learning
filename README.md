# 🇩🇪 Deutschland Karte & Quiz

An interactive geography learning tool for studying German states (Bundesländer), their capitals, and neighboring countries.

## Features

### 📍 Interactive Map
- Real GIS data showing accurate borders of all 16 German states
- Neighboring countries displayed with labels
- Capital cities marked with red dots
- Click on any state to see information
- Clean, simple wireframe design perfect for studying

### 🎯 Quiz Modes

1. **Bundesländer Quiz** - Test your knowledge of German states and their capitals
2. **Nachbarländer Quiz** - Learn Germany's 9 neighboring countries
3. **Gemischtes Quiz** - Mixed questions covering everything
4. **🗺️ Karten-Quiz** - Interactive map quiz where you identify highlighted states

### 🗺️ Map Quiz Mode (Special!)
- The map starts completely empty (white borders only)
- One state at a time is highlighted in yellow
- Answer multiple choice questions about the highlighted state
- Correct answers reveal the state in green with its name and capital
- Watch the map fill in as you progress through all 16 states!

## Technologies Used

- **D3.js** - For rendering accurate geographic boundaries
- **GeoJSON** - Real geographic data for German states and European countries
- **Vanilla JavaScript** - No frameworks, pure JS
- **Responsive Design** - Works on desktop and mobile devices
